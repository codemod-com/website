  ## Description
  This codemod removes unused i18n translations.
  Before:
```jsx
import { useLocale } from '@calcom/lib/hooks/useLocale';

export default function A() {
	const { t } = useLocale();

	return <p>{t('key1')}</p>;
}
```
After:
```jsx
{
	"key1": "key1",
	"key2": "key2"
}
```
